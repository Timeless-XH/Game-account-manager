# 🎮 游戏账号管理器

一个简洁美观的游戏账号管理工具，支持快速索引、一键复制、云端同步。

## ✨ 功能

- **账号管理** — 添加、编辑、删除游戏账号（游戏名、账号、密码、备注）
- **一键复制** — 账号和密码旁边都有复制按钮，点击即可复制到剪贴板
- **智能搜索** — 支持游戏名全名/部分匹配、拼音搜索（全拼、首字母）
- **备注显示** — 有备注内容时显示在游戏名后面的括号里，无备注不显示
- **密码保护** — 应用启动需要输入密码解锁，关闭浏览器后自动锁定
- **云端同步** — 已实现 GitHub Gist 同步，AES-GCM 加密后上传，安全可靠
- **跨设备同步** — 新设备输入 Token 后自动查找已有 Gist，无需重复配置
- **键盘快捷键** — `Ctrl+K` 聚焦搜索、`Ctrl+N` 新建账号、`Esc` 关闭弹窗
- **浅色外观** — 简洁美观的浅色主题，响应式设计，手机也能用

## 📦 使用方式

1. 直接用浏览器打开 `index.html` 即可使用
2. 也可以部署到任意静态服务器（GitHub Pages、Vercel、Netlify 等）

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` / `⌘K` | 聚焦搜索框 |
| `Ctrl+N` / `⌘N` | 新建账号 |
| `Esc` | 关闭弹窗 |

## 🔐 应用密码

启动时需要输入密码才能访问，默认密码：`510052`

- 密码正确后，本次会话保持解锁（关闭浏览器后自动锁定）
- 点击右上角 🔒 按钮可手动锁定应用
- 密码硬编码在代码中，可以自行修改 `APP_PASSWORD` 变量

## ☁️ 云端同步（GitHub Gist）

**首次使用（第一台设备）：**
1. 在 [GitHub Settings → Tokens](https://github.com/settings/tokens) 创建 Personal Access Token，勾选 `gist` 权限
2. 解锁应用后，在底部粘贴 Token，点击「连接 Gist」
3. 自动创建私有 Gist，之后点击「上传」同步数据

**新设备使用：**
1. 解锁应用后，在底部粘贴同一个 Token，点击「连接 Gist」
2. 应用会自动搜索你的 Gist 列表并找到已有数据
3. 点击「下载」即可恢复所有账号数据

## ⚠️ 安全提示

- Token 仅保存在浏览器 localStorage 中，不会上传到任何第三方
- Gist 中的数据经过 AES-GCM 加密，密钥由 Token 派生
- 建议创建 **私有 (private)** Gist
