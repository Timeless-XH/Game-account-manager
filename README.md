# 🎮 游戏账号管理器

一个简洁美观的游戏账号管理工具，支持快速索引、一键复制、云端同步。

## ✨ 功能

- **账号管理** — 添加、编辑、删除游戏账号（游戏名、账号、密码、备注）
- **一键复制** — 账号和密码旁边都有复制按钮，点击即可复制到剪贴板
- **智能搜索** — 支持游戏名全名/部分匹配、拼音搜索（全拼、首字母）
- **备注显示** — 有备注内容时显示在游戏名后面的括号里，无备注不显示
- **密码保护** — 密码默认隐藏，点击眼睛图标切换显示/隐藏
- **本地存储** — 数据保存在浏览器 localStorage，无需后端
- **导出导入** — 支持 JSON 格式备份和恢复
- **云端同步** — 已实现 GitHub Gist 同步，AES-GCM 加密后上传，安全可靠
- **键盘快捷键** — `Ctrl+K` 聚焦搜索、`Ctrl+N` 新建账号、`Esc` 关闭弹窗
- **浅色外观** — 简洁美观的浅色主题，响应式设计，手机也能用

## 📦 使用方式

1. 直接用浏览器打开 `index.html` 即可使用
2. 也可以部署到任意静态服务器（GitHub Pages、Vercel、Netlify 等）

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+K` / `⌘K` | 聚焦搜索框 |
| `Ctrl+N` / `⌘N` | 新建账号 |
| `Esc` | 关闭弹窗 |

## ☁️ 云端同步（GitHub Gist）

1. 在 [GitHub Settings → Tokens](https://github.com/settings/tokens) 创建一个 Personal Access Token，勾选 `gist` 权限
2. 将 Token 粘贴到应用底部的输入框中，点击「连接 Gist」
3. 首次使用会自动创建一个私有 Gist，之后点击「上传」/「下载」即可同步
4. 数据使用 **AES-GCM 加密** 后存储，即使 Gist 泄露也无法读取密码

## ⚠️ 安全提示

- Token 仅保存在浏览器 localStorage 中，不会上传到任何第三方
- Gist 中的数据经过 AES-GCM 加密，密钥由 Token 派生
- 建议创建 **私有 (private)** Gist
